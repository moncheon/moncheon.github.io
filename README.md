# RunMoon - 귀여운 캐릭터 힐링 게임

RunMoon의 공식 웹사이트입니다.

## 게임 소개

시원한 자연 속에서 귀여운 캐릭터들과 함께하는 평화로운 힐링 게임입니다.

### 주요 특징

- 🌳 **평화로운 자연 환경**: 시원한 호수와 푸른 나무들이 있는 힐링 공간
- 🎮 **인터랙티브 캐릭터**: 각진 모서리를 가진 둥근 캐릭터들과 상호작용
- ✨ **간편한 조작**: 캐릭터를 터치하여 원하는 곳으로 이동
- 💚 **힐링 경험**: 일상의 여유와 행복을 선사하는 게임 플레이

## 기술 스택

- HTML5/CSS3
- Vanilla JavaScript
- GitHub Pages

## 웹사이트 구성

- **메인 페이지**: 게임 소개 및 비전 설명
- **인터랙티브 데모**: 캐릭터 움직임을 체험할 수 있는 섹션
- **개인정보 처리방침**: 사용자 정보 보호 정책 (모달 팝업)

## 비전

평화롭고 힐링되는 게임 경험을 통해 일상의 여유와 행복을 선사합니다.
